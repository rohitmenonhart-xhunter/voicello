Voicello Widget Demo

This demo shows how to embed the Voicello widget on any website. The widget appears as a floating button in the corner of your site that users can click to start a video meeting.

How to Embed

Simply add the widget script to your HTML page and initialize it: 

<!-- Add the widget script -->
<script src="https://your-voicello-domain.com/widget/embed.js"></script>

<!-- Optional: Initialize with custom options -->
<script>
  window.addEventListener('load', function() {
    VoicelloWidget.init({
      position: 'bottom-right',
      theme: 'light'
    });
  });
</script>

Or use data attributes for automatic initialization: 

<!-- Add data attributes for auto-initialization -->
<div data-voicello-widget
     data-position="bottom-right"
     data-theme="light"></div>

<!-- Add the widget script -->
<script src="https://your-voicello-domain.com/widget/embed.js"></script>

Configuration Options

Option Default Description
position bottom-right Position of the widget button (bottom-right, bottom-left, top-right, top-left)
theme light Widget theme (light, dark)
widgetUrl Current domain + /widget URL to the widget page
